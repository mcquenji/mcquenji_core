BindConfig<R> repositoryConfig<R extends Repository>()

This function is to be used when binding a Repository to a Module. Use RepositoryInjectorExt.addRepository instead of manually calling this function.

class AppModule extends Module {
  @override
  void binds(i){
    i.addLazySingleton<MyRepo>(MyRepo.new, config: cubitConfig());
  }
}

BindConfig<R> repositoryConfig<R extends Repository>() {
  return BindConfig<R>(
    notifier: (repo) => repo.stream,
    onDispose: (repo) => repo.dispose(),
  );
}
