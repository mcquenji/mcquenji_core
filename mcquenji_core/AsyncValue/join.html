Joins this AsyncValue with another value.

If both this and value are loading, the result will be loading. If either this or value has an error, the result will have that error (this error takes precedence over value's error). Otherwise, the result will be a record of both T and O.

Implementation

AsyncValue<(T, O)> join<O>(AsyncValue<O> value) {
  if (isLoading || value.isLoading) {
    return AsyncValue.loading();
  }

  if (hasError) {
    return AsyncValue.error(error!, stackTrace);
  }

  if (value.hasError) {
    return AsyncValue.error(value.error!, value.stackTrace);
  }

  return AsyncValue.data((requireData, value.requireData));
}
